Build a private password-protected Streamlit UI for subscribed research.

Goal:
- Provide a clean personal app for importing, reviewing, searching, and summarizing subscribed
  Under the Radar material.

Requirements:
- Add simple password gate for the Streamlit app using a configured password hash or secret.
- Do not store plaintext passwords.
- Screens:
  - import PDF/email/text
  - document library
  - latest summaries
  - stock recommendation detail
  - ticker history
  - risks/catalysts/numbers-to-verify
  - source citations
  - download private JSON/Markdown
- Keep UI work-focused and private-use labelled.
- Include visible disclaimer: private research organization only, not financial advice, no
  redistribution.
- Add helper tests for auth/settings and non-Streamlit logic.

Acceptance criteria:
- UI can run locally without live APIs.
- Uploaded synthetic PDFs can be summarized through mock/placeholder paths.
- Password protection is documented.
